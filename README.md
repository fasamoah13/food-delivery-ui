Overview

This project is a mobile UI design for a food delivery application. The goal was to create a clean, modern, and user-friendly interface that allows users to easily browse meals and place orders.

Design Goals

Simple and intuitive navigation
Clean and modern layout
Focus on user experience
Fast ordering flow

Tools Used

Figma (UI Design & Prototyping)
